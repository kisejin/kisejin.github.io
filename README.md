## Getting Started

This is a static website that can be deployed directly to GitHub Pages or any web server.

### Local Development

1. Clone the repository
2. Open `index.html` in your browser

### Customization

- Edit `index.html` to update content (name, description, projects, etc.)
- Modify `css/style.css` to change colors and styling
- Update `js/main.js` to adjust animations and interactions
